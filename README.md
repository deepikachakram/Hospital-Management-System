# Hospital Management System

A console-based Hospital Management System built using Java.

## Features
- Add patient details
- View patient records
- Simple menu-driven interface

## Technologies Used
- Java
- VS Code

## How to Run
1. Open the project in VS Code
2. Compile the files:
   javac src/hospital/*.java
3. Run the program:
   java -cp src hospital.Main
